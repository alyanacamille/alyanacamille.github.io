project write up

My RSA project includes three functions which, encrypt, decrypt, and a welcome message (which includes a while loop). At first this project portrayed itself as easy, with the encryption and creation of keys. However, as time went on I found this project to be a bit more difficult. Implementing the while loop and creating the function for decryption took some time to figure out. As I enjoy doing math, creating the keys was an opportunity for me to use my killer math skills.Yet, a major difficulty I had was decrypting, syntax error continuously popped up preventing me from decrypting, but with the help of my great friend Google, I was able to problem solve and figure out that I had many misspellings, and wrongful indents. This project was independent as I did a vast majority of the project, however my partner was a delight to work with.
